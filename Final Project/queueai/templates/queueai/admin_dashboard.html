Network Access — Second Laptop Setup
Step 1 — Find your host IP:
Windows: Run ipconfig → look for IPv4 Address under your WiFi adapter
Mac/Linux: Run ifconfig | grep inet or ip addr

Step 2 — Start the server on all interfaces:
python manage.py runserver 0.0.0.0:8000

Step 3 — Access from second laptop on same WiFi:
Dashboard: http://192.168.x.x:8000/
Admin panel: http://192.168.x.x:8000/admin

Note: Both laptops must be on the same WiFi network. Replace 192.168.x.x with your actual IP from Step 1.
